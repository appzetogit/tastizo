## Getting Started

### Sign Up / Sign In

1. Open the Tastizo Restaurant app.
2. Choose **Sign In** or **Sign Up**.
3. Sign in with:
   - **Phone** – OTP via SMS
   - **Email** – OTP via email
   - **Google** – Google account

### Onboarding (New Restaurants)

1. Complete **Step 1** – Basic info (name, address, contact).
2. Complete **Step 2** – Menu images, profile image, cuisines, timings.
3. Complete **Step 3** – PAN, GST, FSSAI, bank details.
4. Complete **Step 4** – Display info.
5. Upload documents using **camera** or **gallery** when prompted.

## Orders

### View Orders

1. Go to **Orders**.
2. Use tabs: **Preparing**, **Ready**, **Out for delivery**, **Scheduled**, **Table Booking**, **Completed**, **Cancelled**.

### Order Actions

| Action | When |
|--------|------|
| **Accept** | New order received |
| **Reject** | Cannot fulfill |
| **Preparing** | Start preparation |
| **Ready** | Mark ready for pickup |
| **Resend** | Resend delivery notification |

### Order Details

1. Tap an order.
2. View items, customer, address, payment.
3. **Chat** – Contact customer or delivery partner.
4. **Print** – Print order (if applicable).

### Notifications

- **Sound** – Alert for new orders.
- **Status** – Order status updates.

## Menu Management

### Hub Menu

1. Go to **Explore** → **Menu**.
2. Or **Help Centre** → **Menu** → **Hub Menu**.

### Categories

1. **Add category** – Create new category.
2. **Edit category** – Change name, image.
3. **Reorder** – Drag to reorder.

### Add Menu Item

1. Tap **Add item**.
2. **Images** – Upload via camera or gallery.
3. **Name** – Item name.
4. **Description** – Short description.
5. **Price** – Base price.
6. **Category** – Assign category.
7. **Veg/Non-veg** – Food type.
8. **Variants** – Sizes, add-ons (optional).
9. **Save** – Add to menu.

### Edit Menu Item

1. Tap an item.
2. Edit details.
3. **Save** – Update.

### Add-ons

1. In Hub Menu, open **Add-ons**.
2. Add add-on (name, price, image).
3. Link to menu items.

## Outlet & Settings

### Manage Outlets

1. Go to **Explore** → **Manage Outlets**.
2. Options:
   - **Timings** – Opening/ closing
   - **Contacts** – Owner, manager, staff
   - **FSSAI** – License details
   - **Bank account** – Payout details
   - **Profile picture** – Outlet image
   - **Name, address, location** – Basic info

### Outlet Info

1. Go to **Outlet Info**.
2. Edit:
   - Profile image (camera)
   - Cover images (camera)
   - Name, address
   - Location

### Contact Details

1. Go to **Contact Details**.
2. **Owner** – Edit owner (name, phone, email, photo).
3. **Manager** – Add manager.
4. **Staff** – Add staff.
5. Use **camera** for profile photos.

### Delivery Settings

1. Go to **Delivery Settings**.
2. **Online/Offline** – Toggle availability.
3. Other delivery-related options.

### Outlet Timings

1. Go to **Outlet Timings**.
2. Set opening and closing for each day.
